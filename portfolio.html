MadLibz

Deployed Site GitHub

This is a whimsical web application which allows a user to start a random Mad Libs story, and either enter their own words or create random words based on the parts of speech. After the story is rendered, the user can also save their story as well as view previous saved stories using local storage.

Better Books

Deployed Site GitHub

The Better Books app is fully responsive and designed to allow a user to keep track of the books they have read, want to read, and where they can get good recommendations! The user will also be able to notate their saved books as well as access their saved books from any device with access to the internet.

Daily Planner

Deployed Site GitHub

This is a scheduling application that will allow the user to save events to specific hours on the current day. The save functionality utilises local storage within the browser, and retrieves that stored data when the page reloads for a persistant and reliable schedule. The application utilises moment.js for most of the heavy-lifting, and all for obtaining real time date and time.

Password Generator

Deployed Site GitHub

A random password generation site which will require the user to confirm a series of parameter related questions. Based on the user's responses, the site will generate and print the randomly generated password to text-box.
